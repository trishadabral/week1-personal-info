# Project: Personal Information Program

This repository contains a simple Python program (`personal_info.py`) that demonstrates:

- Storing static information in variables
- Collecting user input with validation
- Displaying formatted output using f-strings
- Testing with different inputs and edge cases
- Adding enhancements like welcome/goodbye messages and age calculation

## Files
- **personal_info.py** → Main Python script with program logic
- **README.md** → Documentation for the project
- **test_inputs.txt** → Sample inputs for testing
- **.gitignore** → Excludes unnecessary files (cache, logs, virtual environments)
